# Overview
Wallet Wise is a personal finance management application designed to help users track their expenses, create budgets, and set financial goals.

# Features
- User authentication and authorization
- Expense tracking and categorization
- Budget creation and management
- Financial goal setting and tracking
- Transaction history and reporting
- User-friendly and intuitive interface

# Requirements
- Java 11 or higher
- Spring Boot 2.5.3 or higher
- Maven or Gradle build tool
- MySQL or PostgreSQL database
- HTML, CSS, and JavaScript for frontend

# Installation
1. Clone the repository using git clone (link unavailable)
2. Navigate to the project directory using cd wallet-wise
3. Build the project using mvn clean package (for Maven) or gradle build (for Gradle)
4. Run the application using java -jar target/wallet-wise.jar (for Maven) or java -jar build/libs/wallet-wise.jar (for Gradle)

# Usage
1. Open a web browser and navigate to http://localhost:8080
2. Click on the "Sign Up" button to create a new account
3. Fill out the registration form and click on the "Register" button
4. Log in to the application using your username and password
5. Start tracking your expenses, creating budgets, and setting financial goals
